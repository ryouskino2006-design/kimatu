#　期末にて提出する予定のプログラムの設計・タスクリスト

##　作成プログラム

### 昼飯
普段自分が行っている昼飯のメニューについて食べられる店・価格・感想や備考をデータとして登録する．

### アニメ
自らの見たことあるアニメをジャンル，評価，感想・備考をデータとして登録する．

###　ゲーム
自らのやったことのあるゲームをジャンル，評価，価格，感想備考をデータとして登録する．



以上のコンセプトを軸に登録，編集，削除，編集，詳細閲覧の機能を持ったプログラムを作成する．

以下をその基本構造とする
```mermaid
graph TD
    A[スタート/ユーザーアクセス] --> B[メニュー一覧];

    %% CRUD - Read (一覧表示/詳細表示)
    B --> C[メニュー項目選択/表示];
    C --> D[メニュー詳細表示];
    D --> B;

    %% CRUD - Update (編集)
    D --> E[編集ボタンを押す];
    E --> F[メニュー編集画面 - Edit];
    F --> G[編集処理実行];
    G --> B;

    %% CRUD - Create (新規登録)
    B --> H[新規登録ボタンを押す];
    H --> I[新規登録画面 - Create];
    I --> J[登録処理実行];
    I --> B; 
    J --> B;

    %% CRUD - Delete (削除)
    B --> K[削除ボタンを押す];
    K --> L[削除処理実行];
    L --> B;
```
