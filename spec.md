# 開発者

システムを三つ掲載する


### ページ遷移の基本構造
```mermaid
graph TD
    A[スタート/ユーザーアクセス] --> B[メニュー一覧];

    %% CRUD - Read (一覧表示/詳細表示)
    B --> C[メニュー項目選択/表示];
    C --> D[メニュー詳細表示];
    D --> B;

    %% CRUD - Update (編集)
    D --> E[編集ボタンを押す];
    E --> F[メニュー編集画面 - Edit];
    F --> G[編集処理実行];
    G --> B;

    %% CRUD - Create (新規登録)
    B --> H[新規登録ボタンを押す];
    H --> I[新規登録画面 - Create];
    I --> J[登録処理実行];
    I --> B; 
    J --> B;

    %% CRUD - Delete (削除)
    B --> K[削除ボタンを押す];
    K --> L[削除処理実行];
    L --> B;
```
